Alarm¶

class Alarm¶

Represents an alarm. Obtained by calling KerbalAlarmClock.alarms, KerbalAlarmClock.alarm_with_name() or KerbalAlarmClock.alarms_with_type().

action¶

Gets or sets the action that the alarm triggers.

Return type:AlarmAction
margin¶

Gets or sets the number of seconds before the event that the alarm will fire.

Return type:double
time¶

Gets or sets the time at which the alarm will fire.

Return type:double
type¶

Gets or sets the type of the alarm.

Return type:AlarmType
id¶

Gets the unique identifier for the alarm.

Return type:string
name¶

Gets or sets the short name of the alarm.

Return type:string
notes¶

Gets or sets the long description of the alarm.

Return type:string
remaining¶

Gets the number of seconds until the alarm will fire.

Return type:double
repeat¶

Gets or sets whether the alarm should be repeated after it has fired.

Return type:bool
repeat_period¶

Gets or sets the time delay to automatically create an alarm after it has fired.

Return type:double
vessel¶

Gets or sets the vessel that the alarm is attached to.

Return type:Vessel
xfer_origin_body¶

Gets or sets the celestial body the vessel is departing from.

Return type:CelestialBody
xfer_target_body¶

Gets or sets the celestial body the vessel is arriving at.

Return type:CelestialBody
delete()¶

Deletes the alarm.
